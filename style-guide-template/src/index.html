Style Guide 1.0

Purpose/Theme

The name of the theme is Starry Skyline. The website will be styled to look like a city skyline silhouette against a starry night sky.
The purpose of thewebsite is to participate in the CSS Zen Garden Challange. It will demonstrate what can be accomplished through CSS-based design.
